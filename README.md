# PipeDrawing
drawing tool for pressure pipes inspection
# 单线图(轴测图)绘制工具

## 1. 目的
让压力管道定期检验和年度检查过程中走线和画图的工作简化.

## 2. 已实现功能
- 通过tkinter和turtle库简单绘制三轴单线图.
- 东南西北上下通过鼠标点击按钮绘制线条.
- 撤销功能.
- 点击按钮绘制阀门和法兰符号.

## 3. 待开发
